# Finalized Portfolio

Simple personal portfolio website to present projects, skills, and contact information.

## Overview

- Clean and responsive layout
- Project showcase
- Skills section
- About section
- Contact section

## Tech Stack

- HTML
- CSS
- JavaScript

## Run Locally

1. Clone the repository:

	```bash
	git clone https://github.com/ItzHipe/Portfolio.git
	```

2. Go to the project folder:

	```bash
	cd Portfolio
	```

3. Open `index.html` in your browser.

Optional (recommended) local server:

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.
